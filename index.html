Overview

The two term adder accepts two terms, sums them, and displays the result

Requirements

  1. The adder is a self-contained web page
  2. The adder has two fields that accept numeric input only
  3. The range of input for each field is 1 through 10, inclusive
  4. Input to the fields can be entered by keyboard or by using the increment/decrement buttons next to each field
  5. On entering a valid input in both fields and activating the 'calculate' button, the sum of the two fields is displayed
  6. The design will be responsive
